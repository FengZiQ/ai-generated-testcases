# 安全测试（Security Testing）

## 适用场景

所有产品，特别是涉及以下功能的产品：
- 用户认证和授权
- 用户数据处理
- 支付/交易
- 文件上传/下载
- API/数据接口
- 用户间交互（UGC）

## 安全测试分类及测试点生成指南

### 1. 认证与身份管理

| 测试领域 | 测试点示例 |
|---------|-----------|
| 密码策略 | "验证密码复杂度要求（大小写、数字、特殊字符的组合策略）" |
| 登录安全 | "验证登录失败次数限制和账户锁定机制" |
| 多因素认证 | "验证已开启2FA时仅密码无法登录" |
| 会话管理 | "验证会话令牌不可预测（无规律可循）" |
| 记住我功能 | "验证'记住我'令牌不会泄露用户凭证" |
| 密码重置 | "验证密码重置链接一次有效且有时效性" |

### 2. 授权与访问控制

| 测试领域 | 测试点示例 |
|---------|-----------|
| 垂直越权 | "验证普通用户无法访问管理员接口" |
| 水平越权 | "验证用户A无法查看/修改用户B的数据" |
| 未授权访问 | "验证未登录状态下无法访问需认证的资源" |
| API权限 | "验证API接口按最小权限原则设计" |
| 目录遍历 | "验证路径遍历攻击被拦截（如../../etc/passwd）" |

### 3. 输入验证与注入

| 测试领域 | 测试点示例 |
|---------|-----------|
| XSS | "验证输入框中的<script>标签被转义而非执行" |
| SQL注入 | "验证SQL注入payload（' OR 1=1 --）被拦截" |
| 命令注入 | "验证命令注入字符（;、|、&&）在输入中被过滤" |
| SSRF | "验证服务器端请求不会访问内部网络地址（127.0.0.1、10.x.x.x）" |
| 文件上传 | "验证上传可执行文件（.exe、.jsp、.php）被拒绝" |
| XML外部实体 | "验证XXE攻击（<!ENTITY xxe SYSTEM>）被拦截" |

### 4. 数据保护

| 测试领域 | 测试点示例 |
|---------|-----------|
| 传输加密 | "验证所有敏感数据传输使用HTTPS/TLS" |
| 存储加密 | "验证密码/令牌等敏感数据以加密形式存储" |
| 敏感信息泄露 | "验证API响应不返回密码、令牌等敏感字段" |
| 日志脱敏 | "验证日志中不记录明文密码、完整银行卡号等" |
| 前端暴露 | "验证前���不暴露API密钥、内部URL等敏感信息" |

### 5. 业务安全

| 测试领域 | 测试点示例 |
|---------|-----------|
| 暴力破解 | "验证API/登录接口有频率限制和防暴力破解机制" |
| 重放攻击 | "验证请求不可重放（含时间戳/nonce）" |
| 并发安全 | "验证高并发下单/扣款无超卖或重复扣款" |
| 优惠滥用 | "验证优惠券/积分不可被脚本批量刷取" |
| 验证码 | "验证图形验证码/滑块验证码不可被自动化绕过" |

### 6. 配置与运维安全

| 测试领域 | 测试点示例 |
|---------|-----------|
| 错误处理 | "验证生产环境不展示堆栈跟踪等调试信息" |
| CORS | "验证跨域策略配置正确（未使用通配符*）" |
| CSP | "验证Content-Security-Policy头已配置" |
| 安全头 | "验证X-Frame-Options、X-Content-Type-Options等安全头已设置" |
| 默认配置 | "验证默认密码/默认端口已修改" |

## 安全测试点生成规则

1. **不依赖需求文档**：安全测试点基于产品类型和功能特征，而非仅从PRD提取
2. **按模块评估**：对每个模块，评估其涉及的安全领域，生成对应测试点
3. **OWASP Top 10参考**：对所有Web产品，参考OWASP Top 10覆盖基础安全测试
4. **严重等级标注**：每个安全测试点标注潜在风险等级

## 安全测试点ID格式

格式：`[MODULE]-SEC-[SEQ]`

- 跨模块的安全测试点，MODULE 使用 SYS
- 从ID可直接识别所属模块
- 示例：
  - SYS-SEC-001：[P0] 验证所有API接口需认证访问
  - AUTH-SEC-001：[P0] 验证SQL注入在登录接口被拦截
  - PYMT-SEC-001：[P0] 验证支付接口有防重放机制

## 安全测试点编写规则

- ✅ "验证用户A无法通过修改URL参数查看用户B的订单"
- ✅ "验证文件上传接口拦截可执行文件（.exe/.jsp/.php）"
- ✅ "验证API响应中不返回用户密码字段"
- ✅ "验证搜索框输入特殊字符（<script>、' OR 1=1--）不被执行"
- ❌ "在搜索框输入<script>alert(1)</script>查看是否弹窗"（含步骤和数据）
- ❌ "检查安全性"（太笼统）
