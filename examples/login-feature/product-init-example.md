# 示例：登录功能产品初始化

## 输入 PRD（简化版）

```
# 登录系统 v1.0

## 概述
提供邮箱密码登录功能，包含账户安全机制。

## 功能模块

### 1. 登录表单
- 邮箱输入框：验证邮箱格式（user@domain.tld）
- 密码输入框：8-64字符
- "登录"按钮：提交表单
- "忘记密码"链接

### 2. 认证逻辑
- 校验凭证匹配性
- 校验账户状态（激活/禁用）
- 连续5次失败锁定15分钟

### 3. 会话管理
- 登录成功生成会话，30分钟无操作过期
- 有操作则重置过期计时器
```

## 初始化输出

### 产品概述

```markdown
# 产品初始化报告

## 产品概述
- 产品名称：登录系统
- 产品类型：Web
- 目标用户：所有系统用户
- 核心价值：提供安全的用户认证

## 模块结构
| 模块ID | 模块名称 | 功能点数量 |
|--------|---------|-----------|
| MOD-001 | 登录表单 | 5 |
| MOD-002 | 认证逻辑 | 4 |
| MOD-003 | 会话管理 | 2 |
```

### 模块详情摘录

**MOD-001：登录表单**
- FP-MOD001-001：邮箱输入验证
  - 输入条件：邮箱字符串
  - 预期：格式正确可提交
  - 异常：格式错误提示"请输入有效邮箱"
- FP-MOD001-002：密码输入验证
  - 输入条件：密码字符串
  - 预期：8-64字符可提交
  - 异常：长度不符提示相应错误

**MOD-002：认证逻辑**
- FP-MOD002-001：凭证验证
- FP-MOD002-002：账户状态验证
- FP-MOD002-003：登录失败计数
- FP-MOD002-004：账户锁定/解锁

**MOD-003：会话管理**
- FP-MOD003-001：会话创建
- FP-MOD003-002：会话过期
