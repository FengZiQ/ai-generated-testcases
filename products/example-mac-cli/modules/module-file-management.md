# 文件管理模块

## 概述

提供文件/目录的复制、移动、删除、搜索、权限管理和链接操作。

## 功能点清单

### 复制功能

**FP-FILE-001：单文件复制**
- **描述**：将源文件复制到目标路径
- **输入条件**：源文件存在、目标路径合法、用户对源有读取权限、对目标目录有写入权限
- **预期行为**：文件内容完整复制到目标路径，元数据（修改时间等）保留
- **异常处理**：源文件不存在→报错"文件不存在"；目标目录无写入权限→报错"权限不足"

**FP-FILE-002：多文件复制**
- **描述**：将多个源文件复制到目标目录
- **输入条件**：多个源文件路径 + 目标目录路径
- **预期行为**：所有文件逐一复制到目标目录
- **异常处理**：任一文件复制失败→继续复制其余文件；全部完成后汇总输出错误列表

**FP-FILE-003：递归目录复制**
- **描述**：使用 --recursive 标志复制整个目录
- **输入条件**：源目录 + 目标路径 + --recursive 标志
- **预期行为**：源目录及其所有子目录和文件递归复制到目标路径
- **异常处理**：源目录不存在→报错；深层子文件无读取权限→跳过该文件并告警

**FP-FILE-004：强制覆盖复制**
- **描述**：使用 --force 标志时直接覆盖已存在的目标文件
- **输入条件**：目标文件已存在 + --force 标志
- **预期行为**：直接覆盖目标文件，无确认提示
- **异常处理**：无特殊异常

**FP-FILE-005：非强制复制遇已存在**
- **描述**：未使用 --force 时复制到已存在的目标
- **输入条件**：目标文件已存在、无 --force 标志
- **预期行为**：提示用户确认是否覆盖
- **异常处理**：用户取消→跳过该文件；非交互模式→跳过

### 移动功能

**FP-FILE-006：单文件移动**
- **描述**：将源文件移动到目标路径
- **输入条件**：源文件存在、目标路径合法
- **预期行为**：文件移动到目标路径，原路径文件被删除
- **异常处理**：跨设备移动→自动采用"复制+删除"策略；目标已存在→同覆盖规则

### 删除功能

**FP-FILE-007：文件删除**
- **描述**：删除指定文件（移入废纸篓）
- **输入条件**：文件存在、用户有写入权限
- **预期行为**：文件被系统移入废纸篓
- **异常处理**：文件不存在→报错；无写入权限→报错

**FP-FILE-008：递归删除目录**
- **描述**：使用 --recursive 标志删除非空目录
- **输入条件**：目录路径 + --recursive 标志
- **预期行为**：目录及其所有内容被删除
- **异常处理**：目录非空且未提供 --recursive→拒绝删除并提示

**FP-FILE-009：强制删除**
- **描述**：使用 --force 标志跳过确认直接删除
- **输入条件**：--force 标志
- **预期行为**：跳过所有确认提示，直接执行删除
- **异常处理**：即使--force也无法删除的无权限文件→报错

### 搜索功能

**FP-FILE-010：按名称搜索文件**
- **描述**：使用 --name 模式搜索文件
- **输入条件**：搜索路径 + --name <pattern>（支持通配符）
- **预期行为**：返回所有匹配模式的文件路径列表
- **异常处理**：无匹配结果→返回空列表；通配符语法错误→报错

**FP-FILE-011：按类型过滤搜索**
- **描述**：使用 --type f/d 过滤文件或目录
- **输入条件**：--type f（文件）或 --type d（目录）
- **预期行为**：仅返回指定类型的条目
- **异常处理**：--type 参数非法（不是 f 或 d）→报错

**FP-FILE-012：按大小过滤搜索**
- **描述**：使用 --size +10M/-1G 过滤文件大小
- **输入条件**：--size +N（大于N）或 --size -N（小于N），单位 K/M/G
- **预期行为**：返回符合大小条件的文件列表
- **异常处理**：大小格式非法（如 --size abc）→报错

**FP-FILE-013：按修改时间过滤搜索**
- **描述**：使用 --mtime +N/-N 过滤修改时间
- **输入条件**：--mtime +N（N天前）或 --mtime -N（N天内）
- **预期行为**：返回符合时间条件的文件列表
- **异常处理**：天数格式非法→报错

### 权限管理功能

**FP-FILE-014：八进制权限修改**
- **描述**：使用八进制模式修改文件权限
- **输入条件**：chmod 755 <file>
- **预期行为**：文件权限修改为 rwxr-xr-x
- **异常处理**：文件不存在→报错；八进制值超出范围(0-777)→报错

**FP-FILE-015：符号权限修改**
- **描述**：使用符号模式修改文件权限
- **输入条件**：chmod u+x/g-w/o=r <file>
- **预期行为**：按符号模式指定的规则修改权限
- **异常处理**：符号语法错误（如 u+z）→报错；操作对象非法→报错

### 链接管理功能

**FP-FILE-016：创建硬链接**
- **描述**：为文件创建硬链接
- **输入条件**：ln <target> <link>
- **预期行为**：创建指向 target 的硬链接
- **异常处理**：target 不存在→报错；跨设备创建→报错

**FP-FILE-017：创建符号链接**
- **描述**：使用 --symbolic 创建符号链接
- **输入条件**：ln -s <target> <link>
- **预期行为**：创建指向 target 的符号链接
- **异常处理**：target 不存在仍可创建（悬空符号链接）；link 已存在→同覆盖规则

**FP-FILE-018：文件信息查看**
- **描述**：查看文件或目录的详细信息
- **输入条件**：文件路径
- **预期行为**：显示文件类型、大小、权限（八进制+符号）、修改时间、所有者/组
- **异常处理**：文件不存在→报错；无读取权限→权限拒绝
