# 网络诊断模块

## 概述

提供网络连通性测试、路由追踪、端口扫描、DNS查询和网络接口信息显示功能。

## 功能点清单

### Ping 功能

**FP-NET-001：基本Ping测试**
- **描述**：对指定主机持续发送ICMP探测包
- **输入条件**：ping <host>（域名或IP地址）
- **预期行为**：持续发送ICMP Echo请求并显示响应，按Ctrl+C停止
- **异常处理**：主机不可达→显示"请求超时"，最终统计丢包率；域名解析失败→报错

**FP-NET-002：指定Ping次数**
- **描述**：使用 --count 指定发送包数量后自动停止
- **输入条件**：ping --count 4 <host>
- **预期行为**：发送4个ICMP包后自动停止，显示统计信息
- **异常处理**：count ≤ 0→使用默认值（持续发送）

**FP-NET-003：指定Ping超时**
- **描述**：使用 --timeout 指定每次探测的超时时间
- **输入条件**：ping --timeout 5 <host>
- **预期行为**：5秒无响应则判定该包超时
- **异常处理**：timeout ≤ 0→使用默认值；值过小（<1）→警告

### 路由追踪功能

**FP-NET-004：基本路由追踪**
- **描述**：显示数据包到达目标所经过的路由
- **输入条件**：traceroute <host>
- **预期行为**：逐跳显示从本地到目标经过的路由器IP和响应时间
- **异常处理**：目标不可达→显示到达最后一跳的信息

**FP-NET-005：指定最大跳数**
- **描述**：使用 --max-hops 限制最大追踪跳数
- **输入条件**：traceroute --max-hops 30 <host>
- **预期行为**：最多追踪30跳，超过则停止
- **异常处理**：跳数 ≤ 0 或 > 255→使用默认值30

### 端口扫描功能

**FP-NET-006：单端口扫描**
- **描述**：检查指定主机的指定端口是否开放
- **输入条件**：portscan <host> --port 80
- **预期行为**：显示端口80的状态（开放/关闭/过滤）
- **异常处理**：主机不可达→报错；端口号超出范围(1-65535)→报错

**FP-NET-007：端口范围扫描**
- **描述**：扫描指定范围内的所有端口
- **输入条件**：portscan <host> --port 1-1024
- **预期行为**：逐一扫描范围内的端口，汇总显示开放端口列表
- **异常处理**：范围格式错误→报错；范围过大（>1000个端口）→提示确认

**FP-NET-008：多端口扫描**
- **描述**：扫描指定端口列表
- **输入条件**：portscan <host> --port 80,443,8080
- **预期行为**：扫描列表中的所有端口
- **异常处理**：端口列表格式错误→报错

### DNS查询功能

**FP-NET-009：A记录查询**
- **描述**：查询域名的IPv4地址记录
- **输入条件**：dnsquery <domain> --type A
- **预期行为**：返回域名对应的所有A记录IP地址
- **异常处理**：域名不存在→返回NXDOMAIN错误

**FP-NET-010：AAAA记录查询**
- **描述**：查询域名的IPv6地址记录
- **输入条件**：dnsquery <domain> --type AAAA
- **预期行为**：返回域名对应的所有AAAA记录
- **异常处理**：域名无AAAA记录→返回空结果

**FP-NET-011：MX记录查询**
- **描述**：查询域名的邮件交换记录
- **输入条件**：dnsquery <domain> --type MX
- **预期行为**：返回邮件服务器及优先级
- **异常处理**：域名无MX记录→返回空结果

### 网络接口功能

**FP-NET-012：显示所有网络接口**
- **描述**：显示系统中所有网络接口的基本信息
- **输入条件**：ifinfo
- **预期行为**：列出所有接口名称、状态、IP地址、MAC地址
- **异常处理**：无

**FP-NET-013：指定接口详情**
- **描述**：使用 --interface 查看指定接口的详细信息
- **输入条件**：ifinfo --interface en0
- **预期行为**：显示指定接口的详细信息（速率、双工模式、MTU、等）
- **异常处理**：接口不存在→报错

### 边界场景

**FP-NET-014：Ping不可达主机**
- **描述**：对不可达地址执行Ping
- **输入条件**：ping 10.255.255.1
- **预期行为**：显示"请求超时"并在结束后显示100%丢包率
- **异常处理**：目标不可达

**FP-NET-015：扫描保留端口0**
- **描述**：对端口0执行扫描
- **输入条件**：portscan <host> --port 0
- **预期行为**：拒绝执行并提示端口号有效范围为1-65535
- **异常处理**：端口0为系统保留端口
