# 系统信息模块

## 概述

提供硬件信息、OS版本、系统日志、用户会话和磁盘使用信息的查看功能。

## 功能点清单

### 硬件信息功能

**FP-SYS-001：显示CPU信息**
- **描述**：查看CPU硬件信息
- **输入条件**：hardware-info --category cpu
- **预期行为**：显示CPU型号、核心数、线程数、频率、缓存大小
- **异常处理**：无（系统调用总可用）

**FP-SYS-002：显示内存信息**
- **描述**：查看内存硬件信息
- **输入条件**：hardware-info --category memory
- **预期行为**：显示内存总容量、类型（DDR4/DDR5）、已用/可用量
- **异常处理**：无

**FP-SYS-003：显示磁盘信息**
- **描述**：查看磁盘硬件信息
- **输入条件**：hardware-info --category disk
- **预期行为**：显示磁盘型号、容量、连接类型（SATA/NVMe/外置）
- **异常处理**：无

**FP-SYS-004：默认显示全部硬件**
- **描述**：无参数时显示所有硬件摘要
- **输入条件**：hardware-info
- **预期行为**：显示CPU+内存+磁盘的摘要信息列表
- **异常处理**：无

### OS版本功能

**FP-SYS-005：显示OS版本**
- **描述**：查看macOS版本信息
- **输入条件**：os-version
- **预期行为**：显示macOS版本号（如14.5）和构建号
- **异常处理**：无

**FP-SYS-006：显示详细OS信息**
- **描述**：使用 --detail 显示更多系统版本信息
- **输入条件**：os-version --detail
- **预期行为**：显示版本号 + 构建号 + 内核版本 + 系统启动时间 + 正常运行时间
- **异常处理**：无

### 系统日志功能

**FP-SYS-007：查看最近系统日志**
- **描述**：显示最近N条系统日志
- **输入条件**：logview
- **预期行为**：按时间倒序显示最近50条系统日志
- **异常处理**：系统日志不可访问→权限拒绝

**FP-SYS-008：按时间过滤日志**
- **描述**：使用 --since 显示指定时间之后的日志
- **输入条件**：logview --since "2026-01-01" 或 logview --since "2h ago"
- **预期行为**：仅显示指定时间之后的日志条目
- **异常处理**：时间格式无法解析→报错并提示可用格式

**FP-SYS-009：按级别过滤日志**
- **描述**：使用 --level 按日志级别过滤
- **输入条件**：logview --level error（可选 error/warn/info/debug）
- **预期行为**：仅显示指定级别及以上的日志
- **异常处理**：级别参数非法→报错并提示可用值

### 用户会话功能

**FP-SYS-010：显示活跃用户会话**
- **描述**：使用 --active 显示当前活跃的用户会话
- **输入条件**：user-sessions --active
- **预期行为**：显示当前登录的用户名、登录时间、终端类型
- **异常处理**：无

**FP-SYS-011：显示所有用户会话**
- **描述**：显示所有用户的登录历史
- **输入条件**：user-sessions
- **预期行为**：按时间倒序显示登录记录（用户、终端、登录/登出时间、来源IP）
- **异常处理**：无

### 磁盘使用功能

**FP-SYS-012：查看指定目录磁盘使用**
- **描述**：计算并显示指定目录的总大小
- **输入条件**：disk-usage <path>
- **预期行为**：以字节为单位显示目录总大小和包含的文件/目录数量
- **异常处理**：路径不存在→报错；无读取权限→权限拒绝

**FP-SYS-013：可读格式显示**
- **描述**：使用 --human-readable 以K/M/G单位显示
- **输入条件**：disk-usage <path> --human-readable
- **预期行为**：自动选择合适的单位（KB/MB/GB/TB）显示
- **异常处理**：同FP-SYS-012

**FP-SYS-014：查看根目录磁盘使用**
- **描述**：查看根目录磁盘使用情况
- **输入条件**：disk-usage /
- **预期行为**：显示根目录总大小
- **异常处理**：无

### 统计功能

**FP-SYS-015：查看网络统计信息**
- **描述**：使用 ifinfo --statistics 显示网络统计数据
- **输入条件**：ifinfo --statistics
- **预期行为**：显示各接口收发数据包总数、错误数、丢弃数
- **异常处理**：无
