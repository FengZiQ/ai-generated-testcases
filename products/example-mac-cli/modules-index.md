# Mac CLI Toolkit 模块索引

## 模块总览

| 模块ID | 模块名称 | 缩写 | 功能点数 | 依赖模块 | 依赖外部系统 |
|--------|---------|------|---------|---------|------------|
| MOD-001 | 文件管理 | FILE | 18 | 无 | 文件系统 |
| MOD-002 | 进程管理 | PROC | 15 | 无 | 进程API |
| MOD-003 | 网络诊断 | NET | 15 | 无 | Network.framework |
| MOD-004 | 系统信息 | SYS | 15 | 无 | IOKit, 系统框架 |

## 依赖关系图

```
┌──────────────────────────────────────────────────┐
│                 Mac CLI Toolkit                    │
├──────────┬──────────┬──────────┬──────────────────┤
│  FILE    │  PROC    │  NET     │      SYS         │
│ (独立)   │ (独立)   │ (独立)   │    (独立)         │
├──────────┴──────────┴──────────┴──────────────────┤
│              macOS 系统层 API                       │
│   (Foundation / Network.framework / IOKit / ...)   │
└──────────────────────────────────────────────────┘
```

> 四个模块均为独立模块，彼此无功能依赖。每个模块直接调用 macOS 系统 API。

## 功能点分布

| 模块 | EQP适用 | BVA适用 | DCS适用 | STT适用 | SEC适用 | PERF适用 | CMP适用 |
|------|---------|---------|---------|---------|---------|---------|---------|
| FILE | ✓ | ✓ | ✓ | - | ✓ | ✓ | - |
| PROC | ✓ | ✓ | - | ✓ | ✓ | ✓ | - |
| NET | ✓ | ✓ | - | - | ✓ | ✓ | ✓ |
| SYS | ✓ | - | - | - | ✓ | ✓ | ✓ |

## 外部系统集成点

| 集成点 | 涉及模块 | 说明 |
|--------|---------|------|
| macOS 文件系统 | FILE | 所有文件操作通过系统文件API |
| macOS 进程API | PROC | 进程列表和信号通过系统进程API |
| Network.framework | NET | 网络操作通过系统网络框架 |
| IOKit | SYS | 硬件信息通过IOKit框架获取 |
| 系统日志（os_log） | SYS | 日志查看通过系统日志API |
